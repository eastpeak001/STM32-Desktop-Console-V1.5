# Assembly Notes

## 1. 焊接顺序

推荐焊接顺序：

1. 电源模块
2. 电源指示 LED
3. STM32 核心板插座
4. 按键和 EC11
5. 模块排母
6. LED 和蜂鸣器接口
7. 其他接口

## 2. 上电前检查

用万用表蜂鸣档检查：

- 5V_IN 与 GND 不应短路
- 5V_BUS 与 GND 不应短路
- 3V3 与 GND 不应短路

## 3. 首次上电

只焊接电源部分时先上电测试：

- 输入 5V
- 测量 5V_BUS
- 测量 3V3
- 确认 AMS1117 不异常发热

## 4. 模块插入顺序

建议按以下顺序插入并测试：

1. STM32 核心板
2. OLED
3. AHT20
4. 光敏模块
5. MAX7219
6. BT24
7. W25Q64
8. MPU6050
9. 蜂鸣器模块

## 5. 常见问题

### OLED 不显示

检查：

- VCC 是否为 3.3V
- GND 是否正确
- SCL/SDA 是否接反
- I2C 地址是否正确

### 蓝牙无回复

检查：

- BT24 是否使用 3.3V 供电
- PA9/PA10 是否交叉连接
- 手机发送命令是否带换行符
- UART 波特率是否一致

### I2C 设备不稳定

检查：

- 模块是否自带上拉
- 是否需要焊接板上 4.7k 上拉电阻
- SCL/SDA 是否接反

### 板子上电短路

检查：

- 3V3 与 GND 是否短接
- 5V_BUS 与 GND 是否短接
- AMS1117 引脚方向是否正确
- 模块是否反插
