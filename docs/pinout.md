# Pinout

## STM32F103C8T6 Pin Assignment

| STM32 引脚 | 网络名 | 功能 |
|---|---|---|
| PA0 | PA0_ADC_LIGHT | 光敏 ADC 输入 |
| PA1 | PA1_ENC_A | EC11 A 相 |
| PA2 | PA2_ENC_B | EC11 B 相 |
| PA3 | PA3_KEY | OK 按键 |
| PA4 | PA4_FLASH_CS | W25Q64 CS |
| PA5 | PA5_SPI1_SCK | SPI1 SCK |
| PA6 | PA6_SPI1_MISO | SPI1 MISO |
| PA7 | PA7_SPI1_MOSI | SPI1 MOSI |
| PA9 | PA9_USART1_TX | USART1 TX / BT24 RXD |
| PA10 | PA10_USART1_RX | USART1 RX / BT24 TXD |
| PB6 | PB6_I2C1_SCL | I2C1 SCL |
| PB7 | PB7_I2C1_SDA | I2C1 SDA |
| PB8 | PB8_MAX_DIN | MAX7219 DIN |
| PB9 | PB9_MAX_CS | MAX7219 CS |
| PB10 | PB10_MAX_CLK | MAX7219 CLK |
| PB12 | PB12_LED_USER1 | 用户 LED1 |
| PB13 | PB13_LED_USER2 | 用户 LED2 |
| PB14 | PB14_LED_BT_TX | 蓝牙发送指示 LED |
| PB15 | PB15_BUZZER | 蜂鸣器控制 |
| NRST | NRST | 复位按键 |
